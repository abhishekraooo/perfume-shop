# Perfume Shop - Full Stack Assessment

## Overview
A responsive, full-stack e-commerce application built with the MERN stack (MongoDB, Express, React, Node.js). Features include a custom-built API, local image serving, interactive gallery, and responsive design.

## Features Implemented
- **Dynamic Hero Section:** CSS Keyframe animations and Marquee strip.
- **Product Catalog:** Filtering logic (Category) and Responsive Grid.
- **Advanced Product Details:** Sticky layout, Olfactory notes, and Interactive Gallery.
- **Backend:** Node/Express server with MongoDB Atlas integration.
- **Simulation:** Google Login UI mock-up.

## How to Run

### 1. Backend (Server)
1. Navigate to the `server` folder: `cd server`
2. Install dependencies: `npm install`
3. Start the server: `node server.js`
   - Runs on: http://localhost:5000
   - **Note:** Images are served locally from `server/public/images`.

### 2. Frontend (Client)
1. Navigate to the `client` folder: `cd client`
2. Install dependencies: `npm install`
3. Start the app: `npm run dev`
   - Runs on: http://localhost:5173

## Tech Stack
- Frontend: React.js, CSS3 (Grid/Flexbox)
- Backend: Node.js, Express.js
- Database: MongoDB Atlas (Cloud)
